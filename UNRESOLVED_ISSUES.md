# مسائل و محدودیت‌های باقی‌مانده

هیچ مسئلهٔ Critical یا High شناخته‌شده‌ای در دامنهٔ اختیار WorkGPT باز نمانده است. موارد زیر blocker تولید نیستند، اما باید در بازرسی مستقل و انتشار دستی صریح بمانند.

| موضوع | وضعیت و اقدام بعدی |
|---|---|
| بازرسی انسانی وب | WorkGPT ساخت، لینک، responsive، keyboard، axe و interaction را آزموده است، اما تصویب زیبایی‌شناختی انسانی وب ادعا نمی‌شود؛ Codex باید همهٔ ۱۴ صفحه را روی موبایل و دسکتاپ ببیند. |
| WebKit | در خودممیزی اجرا نشده است؛ Codex باید در صورت دسترسی Safari/WebKit را به regression اضافه کند یا محدودیت را گزارش دهد. |
| Firefox موبایل | درخواست ۳۹۰px در محیط headless حاضر client width برابر ۴۳۸px می‌دهد؛ عرض‌های دقیق ۳۶۰ و ۳۹۰ در Chromium عبور کرده‌اند. |
| XeLaTeX میزبان | Quarto/Pandoc/XePersian pin هستند، اما TeX Live کامل hermetic بسته‌بندی نشده است؛ میزبان باید XeLaTeX سازگار با baseline 2023 داشته باشد. |
| design debt مستقل | عیب‌های روشن whitespace و hierarchy در Phase 5 اصلاح و PDF بازبینی شد؛ Codex همچنان باید cohesion فاصله، جدول‌های bare، print CSS و consistency بین ماژول‌ها را مستقل نقد کند. |
| پیوندهای بیرونی | ۳۹ مقصد bibliographic ثبت و هنگام تولید معتبر بودند؛ availability آیندهٔ وب تضمین‌شدنی نیست. build آموزشی اصلی برای محتوا یا رسانهٔ critical به hotlink وابسته نیست. |
| رسانهٔ محدود | فقط سه رسانهٔ دارای حق روشن پذیرفته شد. این کمبود عمدی و سازگار با سیاست کیفیت/حق است؛ Codex باید فقط نیاز آموزشی حل‌نشده را گزارش کند، نه برای رسیدن به سهمیه شکل اضافه کند. |
| انتشار remote | شاخه و کامیت‌ها فقط محلی‌اند. مالک باید bundle را وارد و پس از بررسی دستی push کند؛ WorkGPT هیچ write به GitHub انجام نداده است. |
| نتیجهٔ Codex | `CODEX_AUDIT_REPORT.md` هنوز وجود ندارد و audit اجرا نشده است. آماده‌بودن برای audit به معنی independently audited بودن نیست. |

پنج هشدار `Underfull` در log نهایی XeLaTeX به‌ازای هر pass به خط‌های کوتاه واژه‌نامه مربوط‌اند؛ در pass نهایی ۱۵ رخداد ثبت شد. آن‌ها clipping یا overflow نیستند و صفحه‌های مربوط دستی خوانده شدند. Codex می‌تواند اثر بصری را گزارش یا spacing کم‌خطر را اصلاح کند، اما نباید برای صفرکردن warning تعریف‌های واژه‌نامه را بی‌دلیل بازنویسی کند.
